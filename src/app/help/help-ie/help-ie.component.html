This app/tool has been packaged with a very powerfull and full-featured inline editor component. This editor can be used to edit/fix the generated scripts without having to go through the hassle of downloading the script and then downloading a 3-rd party tool for editing the script. This editor component will make it easy for the user to edit the script and then save/persist the changes in no-time This tool can be accessed from two sreens, Upload Mapping and View/Edit history, under the result section of the respective screens as shown in img 1.1 and img 1.2
