There are three different ways for exporting the generated scripts
Export As Excel Export As Single SQL File Export As Multiple SQL Files

This option enables the user to export the generated scripts as an excel spreadsheet for futher fixing and editing. A single excel file will be generated and downloaded as shown in img 1.1

img 1.1

This option enables the user to export the generated scripts as a SQL file for futher fixing and editing. A single SQL file will be generated and downloaded as shown in img 2.1

img 2.1

This option enables the user to export the generated scripts as Multiple SQL files for futher fixing and editing. There will be a .sql file for each import tables in the mapping. A Zip file will be generated and downloaded as shown in img 3.1. In case of multiple inserts for a single import table, the generated sql file will be appened with a random number.
