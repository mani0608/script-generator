This app/tool maintains/persists the history of each individual mapping uploads. In case if the user wants to edit or download any of the previous mapping upload version, he can use the filter shown in img 1.1 to filter the version he wants to work on and then edit or export the scripts. This option is disabled in case of no upload history.

Filter History View Script Edit Script Export Results
