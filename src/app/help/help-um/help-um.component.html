Additionally this page provides the following features to the user
View Script Edit Script Export Results

Once filtering the upload, user can use the view icon to the right in each of the row to view the respective script as shown in img 2.1. In order to view all the scripts under a group (source table) click on the view icon in the header of respective group (source table) as shown in img 2.2

Once filtering the upload, user can use the view icon to the right in each of the row to edit the respective script and double click the cell highlighted in blue to activate the inline editor as shown in img 3.1.
