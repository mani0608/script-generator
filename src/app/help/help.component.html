This app accepts a single mapping spreadsheet through the dropzone upload component. It will then parse the mapping document and generates different insert/import scripts for each import table mapping. This page will provide details on how to use this app and will expose you to the aspect of how certain option/feature works

This app/tool currently offers following functionalities. Please click on the heading to navigate to the respective sections
Upload & Edit Mapping
Export Scripts
View/Edit upload history (with version)
SQL Inline editor
Query Builder (In progress)
