Unpack the zip file to a folder of your choice. Please don't extract it directly under 'C' or 'D' drive.
Double click the "script-generator.exe" file
The application will open multiple times, but please wait for installer loading screen to disappear.
Once the installer completes, you can start using the tool. You can find the installed application under C:\Users\...\AppData\Local\script-generator.exe
